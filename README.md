# Tunç Türker - Personal Website

A premium, minimal personal website built with HTML, CSS, and Vanilla JavaScript.

## Features
- **Dynamic Content**: Uses local JSON/JS data for Films and Thoughts feeds.
- **Micro-blogging**: Standalone essay pages with automated summary cards.
- **Responsive Design**: Mobile-friendly with smooth scroll animations.
- **Premium Aesthetics**: Glassmorphism, blurred navigation, and curated typography.

## Tech Stack
- HTML5 / CSS3 (Vanilla)
- Vanilla JavaScript
- Font: [Inter](https://fonts.google.com/specimen/Inter), [Crimson Pro](https://fonts.google.com/specimen/Crimson+Pro)
